# Finland deep-dive sources

## Starting point from the local dossier

The existing Finland dossier was already strong on the national model. It correctly described Finland as a TENK-led self-regulatory system in which local research organisations handle alleged research integrity violations first, and TENK gives statements only after a completed local process.

The case-file question for this pass was therefore narrower:

- whether TENK publishes full statements or only summaries
- how far the local first-instance files are public
- whether local public-document status is the same as proactive case publication
- whether TENK's human-sciences ethics statement route should be separated from the core RI statement route

## Key official source groups checked in this pass

- TENK statement summaries page
- TENK request-for-statement page
- TENK Annual Report 2024

- TENK committed-organisations list
- Aalto University handling of RI misconduct page
- University of Helsinki RI page
- University of Jyvaskyla RI page

## Key conclusions for coding

- Finland has a real structured national archive, but it is a searchable archive of anonymized summaries, not a full statement or full decision archive.
- Local research organisations remain the first-instance handlers. TENK only reviews the process or RI question after a local preliminary inquiry or investigation proper.
- TENK states that it does not publish its statements; only case summaries are published online.
- The request-for-statement route says the request, appendices and processing documents become public after the statement, except confidential information. This is a public-document-access layer, not the same as proactive online publication.
- Aalto is a useful local example because it explicitly says all documents relating to a misconduct case are public, while still not maintaining a public case archive.
- TENK can return a matter to the organisation for an investigation proper when a possible RI violation cannot be excluded.
- TENK also issues statements on human-sciences ethical review disputes. That is a boundary statement lane and should not be merged with the RI statement-summary archive.

### Method notes from the pilot

- Public-document status is not the same as proactive case publication. Finland needs a separate `publicity mode` field.
- A structured archive can be a summary archive rather than a full decision or full statement archive. Finland needs a separate `publication depth` field.
- In self-regulatory countries, code the first-instance organisation and the national review/statement body separately even if the national body owns the best public archive.
- Boundary statement lanes should be separated when the same national body also publishes or summarizes ethics-review disputes.
